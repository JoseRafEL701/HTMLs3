# 📘 HTMLs3 - Learn HTML from Scratch

This repository contains the project developed as part of my learning in **HTML5** and **CSS3**. The main goal is to create an educational website that serves as support material to learn the basic and fundamental concepts of the HTML language.

---

## 🎯 Project Objective

To develop a complete, well-structured, and responsive website that allows users to learn the essential elements of HTML5 in a clear and practical way, while applying good web development practices.

Through this page, the aim is to reinforce the understanding of:
- The correct structure of an HTML document
- The proper use of semantic tags
- Content organization and layout
- Basic principles of responsive design

---

## 🛠️ Technologies Used

- **HTML5** - Structure and semantic content
- **CSS3** - Styles, design, and responsiveness (Responsive Design)
